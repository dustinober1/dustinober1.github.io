EduSched
Open Source Library
Python Package

A powerful constraint-based scheduling library designed specifically for educational institutions to create optimal class schedules while respecting complex constraints.

  • Constraint Solving: Hybrid approach offering both fast heuristic algorithms and optimal solutions via Google OR-Tools.
  • Domain Specific: Rich data model handling rooms, instructors, students, and hard/soft constraints (e.g., availability, conflicts).
  • Flexible Integration: Pure Python library with optional FastAPI backend support for web integrations.
ChainReaction
AI Agents & Graph Architecture
Multi-Agent System

An autonomous, AI-powered supply chain risk monitoring platform that combines real-time news analysis, graph-based impact tracing (GraphRAG), and multi-agent orchestration.

  • Multi-Agent Orchestration: Utilizes LangGraph and DSPy to coordinate specialized agents that autonomously scout news, extract risk signals, and assess confidence.
  • GraphRAG Engine: Leverages Neo4j to map complex supply chain dependencies and trace multi-hop impacts of disruptions in real-time.
  • Local-First AI: Supports fully local execution using Ollama, integrated with a Next.js visualization dashboard and FastAPI backend.
UI Heist
Developer Tools & Productivity
Chrome Extension

A powerful Chrome extension that allows developers to "steal" (inspect) UI elements from any website and instantly convert them into ready-to-use React components.

  • Instant Conversion: Click any element to generate clean React/JSX code with computed inline styles (typography, spacing, colors).
  • Workflow Acceleration: Bridges the gap between inspiration and implementation, allowing devs to rapidly prototype using existing web designs.
  • Tech Stack: JavaScript, Chrome Extensions API, HTML/CSS.
